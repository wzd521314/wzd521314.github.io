基础知识

  1. 什么是容器，什么是项目，什么是主轴，什么是侧轴？
  2. 项目永远排列在主轴上
  3. flex分新旧两个版本：-webkit-box和flex

注意点：我们都知道新版本的flex要比老版本的flex强大很多，有没有必要学习老版本的flex?
答：移动端开发者必须关注老版本的flex,因为很多移动端设备内核低，只支持老版本的flex。

老版本的box通过两个属性，四个属性值控制了主轴的位置和方向
新版本的flex通过一个属性，四个属性值控制了主轴的位置和方向

富裕空间的管理只决定富裕空间的位置，不会给项目分配空间

flex(老版本) display:-webkit-box

容器
容器的布局方向: 控制主轴和侧轴分别是哪一根 -webkit-box-orient

  • horizontal 主轴为x轴
  • vertical 主轴为y轴

容器的排列方向：控制主轴方向 -webkit-box-direction

  • normal 从左往右(正向)
  • reverse 反向

富裕空间管理(主轴)-webkit-box-pack

  • start 富裕空间在右边(x轴)，下边(y轴)
  • end 富裕空间在左边(x轴)，上边(y轴)
  • center 富裕空间在两边
  • justify 富裕空间在项目之间

富裕空间管理(副轴)-webkit-box-align

  • start 富裕空间在右边(x轴)，下边(y轴)
  • end 富裕空间在左边(x轴)，上边(y轴)
  • center 富裕空间在两边

项目
弹性空间的管理:将主轴上的富裕空间按比例分配到项目上 -webkit-box-flex:1

flex(新版本) display:flex

容器
容器的布局方向：控制主轴和侧轴分别是哪一根 flex-direction

  • row 主轴为x轴
  • column 主轴为y轴

容器的排列方向：控制主轴方向 flex-direction:column-reverse

  • normal 正向
  • reverse 反向

富裕空间管理(主轴) justify-content

  • flex-start 富裕空间在主轴的正方向
  • flex-end 富裕空间在主轴的反方向
  • center 富裕空间在两边
  • space-between 富裕空间在项目之间
  • space-around 富裕空间在项目两边

富裕空间管理(副轴) align-items

  • flex-start 富裕空间在侧轴的正方向
  • flex-end 富裕空间在侧轴的反方向
  • center 富裕空间在两边
  • baseline 按基线对齐
  • stretch 等高布局(项目没有高度时)

项目
弹性空间的管理:将主轴上的富裕空间按比例分配到项目上 flex-grow：1

flex容器新增

flex-wrap:该属性控制了容器为单行/列还是多行/列。并且定义了侧轴的方向。新行/列将沿侧轴方向堆砌

  • nowrap 默认值为单行
  • wrap 多行(flex-shrink失效)
  • wrap-reverse 多行且反向(flex-shrink失效)

align-content:控制多行多列时富裕空间的管理，会把所有行、列看作一个整体

  • flex-start 富裕空间在侧轴的正方向
  • flex-end 富裕空间在侧轴的反方向
  • center 富裕空间在两边
  • baseline 按基线对齐
  • stretch 等高布局(项目没有高度时)

flex-flow:属性是是设置flex-direction和flex-wrap的简写

flex项目新增

order：0 属性规定了弹性容器中的可伸缩项目在布局时的顺序.越大越后

align-self 项目自身富裕空间的管理(优先级高)

  • flex-start 富裕空间在侧轴的正方向
  • flex-end 富裕空间在侧轴的反方向
  • center 富裕空间在两边
  • baseline 按基线对齐
  • stretch 等高布局(项目没有高度时)

flex-shrink 收缩因子（默认值为1）
flex-basis 伸缩规则计算的基准值（默认拿width或height的值）

伸缩规则

flex-basis（默认值为auto）
flex-grow （默认值为0）

可用空间 = (容器大小 - 所有相邻项目flex-basis的总和)
可扩展空间 = (可用空间/所有相邻项目flex-grow的总和)
每项伸缩大小 = (伸缩基准值flex-basis + (可扩展空间 x flex-grow值))

flex-shrink （默认值为1）

计算收缩因子与基准值乘的总和 var a = 每一项flex-shrinkflex-basis之和
计算收缩因数 收缩因数=（项目的收缩因子项目基准值）/第一步计算总和 var b = (flex-shrink*flex-basis)/a
移除空间的计算 移除空间= 项目收缩因数 x 负溢出的空间 var c = b * 溢出的空间

flex简写属性

flex: flex-grow flex-shrink flex-basis
等分布局 flex:1

响应式布局方案

媒体查询选择器 @media XXX{}

媒体类型

  • all
  • screen
  • print

媒体属性

  • width：浏览器的窗口尺寸
  • device-width：设备尺寸
  • device-pixel-ratio（必须加webkit前缀）：像素比

-以上三个媒体属性可加 min 和 max 前缀 min：最小值为谁 max：最大值为谁
横竖屏切换

  • orientation：landscape（横屏）
  • orientation：portrait （竖屏）

关键字

  • and:代表与的意思，一般用and来连接媒体类型和媒体属性
  • only:和浏览器兼容性有关。老版本的浏览器只支持媒体类型，不支持带媒体属性的查询
  • ,代表或的意思，

多列布局

  • column-width 栏目宽度
  • column-count 栏目列数
  • column-gap 栏目距离
  • column-ruler 栏目间隔线
